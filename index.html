What is Machine Learning?

You have most likely heard of machine learning by now but you may not know what it really is. Machine learning is a term used in Computer Science to describe a computer program or algorithm that has the ability to learn. Machine learning has many different applications from artificial intelligence to big data analysis. Deep learning, a type of machine learning, is now even used to power Google's search engine1.

There are many different types of machine learning all with different pros and cons. In this article I am going to cover how to use machine learning for data analysis and for artificial intelligence. I will provide code samples along the way so make sure you have Python 2.7 installed if you would like to follow along. If you want to work with Python in an isolated environment to ensure you do not have conflicting packages see virtual environments. I will try to make this as easy to follow for those without knowledge of programming.

Machine Learning in Data Analysis

As technology becomes insreasingly integrated with our everyday life, telemetry finds its way into more and more of the software we use everyday. Companies like Microsoft, Amazon and Google record massive amounts of data through tracking cookies planted on websites and usage data reported from their software. To uncover valuable trends and information hidden within these large data sets, data researchers use machine learning algorithms to analyze and make sense of the data.

I am going to show you how you can use machine learning with Python to analyze data and find trends using the same algorithms used by Big Data.

To begin lets setup our work environment. Make a new directory for the project. If you are going to setup a virtual environment you should do so now. Run the following command to install the dependencies needed for this project.

$ pip install scipy numpy matplotlib pandas sklearn

Import all the needed libraries with the following code

import pandas
from pandas.tools.plotting import scatter_matrix

from sklearn.naive_bayes import GaussianNB

We will be working with the iris flowers dataset. This dataset is often used as a hello world example for machine learning. You can find the dataset here. Use the following code to load the data into a labeled dataset.

url = "https://archive.ics.uci.edu/ml/machine-learning-databases/iris/iris.data"
names = ['sepal-length', 'sepal-width', 'petal-length', 'petal-width', 'class']
dataset = pandas.read_csv(url, names=names)

Before feedind the data into our model we will split it into two arrays. The first array will be used to train the model and will consist of the first 80% of the iris data set. The other 20% of the data will be used to test the model. After training the model with the first 80%, we will have it generate data and see how close it is to the other 20%. The following code splits the data like so. We set the scoring variable to accuracy which determines how we will determine the best algorirthm. The seed is used for the random number generator and is there just to make sure that we both get the same results.

# Split-out validation dataset array = dataset.values X = array[:,0:4] Y = array[:,4] validation_size = 0.20 seed = 72 scoring = 'accuracy' X_train, X_validation, Y_train, Y_validation = model_selection.train_test_split(X, Y, test_size=validation_size, random_state=seed)

Now that we have loaded the data we are ready to run it through the learning algorithms. We will use 6 different algorithms and assess their accuracy to choose the best model for our data. After running the following code you should see it output the score or fitness for each algorithm. You should use this value to choose the best algorithm for your data.

models = []
models.append(('LR', LogisticRegression()))
models.append(('LDA', LinearDiscriminantAnalysis()))
models.append(('KNN', KNeighborsClassifier()))
models.append(('CART', DecisionTreeClassifier()))
models.append(('NB', GaussianNB()))
models.append(('SVM', SVC()))
results = []
names = []
for name, model in models:
kfold = model_selection.KFold(n_splits=10, random_state=seed)
cv_results = model_selection.cross_val_score(model, X_train, Y_train, cv=kfold, scoring=scoring)
results.append(cv_results)
names.append(name)
msg = "%s: %f (%f)" % (name, cv_results.mean(), cv_results.std())
print(msg)

After choosing which algorithm you want to go with, you can use it to make predictions for your data. In this example I chose the KNN algorirthm to predict possible values that could be included in the data set.

knn = KNeighborsClassifier()
knn.fit(X_train, Y_train)
predictions = knn.predict(X_validation)
